句子情感分析是自然语言处理领域中的重要任务，可挖掘文本
情感倾向，为舆情监测等提供基础。本项目基于 PaddlePaddle 和 Scikitlearn 框架，实现了三种情感分类模型：TextCNN、决策树和逻辑回归，
使用千言 ChnSentiCorp 酒店评论数据集进行对比实验。TextCNN 通
过卷积神经网络捕捉局部特征；决策树基于信息增益构建分类规则；逻
辑回归利用线性决策边界实现高效分类。实验结果表明：TextCNN 效
果最佳（测试集准确率 0.9167），决策树可解释性最强，逻辑回归训练
速度最快。多模型对比验证了深度学习在特征提取上的优势，同时为
不同应用场景提供技术选型参考。
